# USG-Kuwait


USG Kuwait — Construction Company Portfolio Website
A professional portfolio website built for United Solutions Group (USG), a Kuwaiti construction company founded in 2007 specializing in mosques, villas, and commercial buildings.
Live site: usg-kuwait.vercel.app

About the Project
This was built as a freelance project — a fully custom, non-template website designed to showcase USG's completed construction projects across Kuwait. The design is inspired by high-end architecture firm websites, featuring full-bleed photography, cinematic transitions, and a minimal dark aesthetic.

Features

Cinematic project gallery with fade and zoom transitions
21 real completed projects with construction phase and final photos
Category filtering — Mosques, Villas, Commercial
English / Arabic bilingual toggle with RTL layout support
Floating WhatsApp contact button
Google Maps office location integration
Fully mobile responsive
SEO optimized and indexed on Google via Search Console


Tech Stack

React + Vite + TypeScript
Tailwind CSS
Vercel (deployment)
Google Search Console (SEO)


Built With
Developed using Claude Code for AI-assisted development, with Google Stitch used for initial layout generation.
